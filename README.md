# 🐍 Smart Store Billing System

A simple console-based billing system built using Python.  
This project simulates a basic store where users can view products, manage a cart, and generate a bill with GST.

---

## 🚀 Features

- 📋 Menu-driven interface
- 🛒 Add products to cart
- ❌ Remove products from cart
- 📦 Real-time stock management
- 💰 Bill generation with Subtotal + GST (18%) + Final Total

---

## 🧠 Concepts Used

- Python Dictionaries (Nested)
- Loops (while, for)
- Conditional Statements (if-elif-else)
- User Input Handling
- Basic Billing Logic
1.View Products
2.Add Product to Cart
3.Remove Product from Cart
4.Generate Bill & Exit
